## Команды разработки

```bash
# Установка зависимостей
npm install          # или: pip install -r requirements.txt

# Запуск в dev-режиме
npm run dev          # или: python manage.py runserver

# Запуск тестов
npm test             # или: pytest

# Сборка
npm run build        # или: python setup.py build

# Линтер
npm run lint         # или: flake8 .
```
