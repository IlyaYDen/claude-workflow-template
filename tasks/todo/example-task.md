# Задача: Добавить кнопку выхода из аккаунта

## Описание

Добавить кнопку "Выйти" в хедер приложения для авторизованных пользователей.

## Требования

### Функциональные
- Кнопка отображается только для авторизованных пользователей
- При клике — выход из аккаунта и редирект на главную
- Очистка токена авторизации из localStorage/cookies

### UI/UX
- Расположение: правый угол хедера
- Текст: "Выйти" или иконка выхода
- Подтверждение не требуется (выход сразу)

### Технические
- Использовать существующий AuthService
- Не ломать текущую навигацию
- Работает на всех страницах

## Критерии приёмки

- [ ] Кнопка видна только авторизованным
- [ ] Клик очищает сессию
- [ ] Редирект на главную работает
- [ ] Тесты написаны и проходят

## Дополнительно

- Дизайн: согласовать с существующими кнопками в хедере
- Приоритет: средний
- Связанные задачи: нет
